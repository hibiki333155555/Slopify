# End-User Install Guide

## Goal
Install and run the Slopify desktop app without cloning source code.

## 1. Download
1. Open the latest release page for your organization build artifacts.
2. Download the installer for your OS:
- macOS: `.dmg`
- Windows: `.exe`
- Linux: `.AppImage` or `.deb`

## 2. Install
1. Run the installer package.
2. Launch `Slopify` from Applications / Start Menu.

## 3. First-run setup
1. Enter `Display name`.
2. Optionally enter `Avatar URL`.
3. Enter `Server URL` (example: `https://sync.example.com`).
4. Enter `Server access password` provided by your admin.
5. Select `Continue`.

## 4. Join or create project
1. Use `Create project` for a new workspace.
2. Or paste an invite code in `Join with invite`.

## Notes
- Data is local-first: your events are saved locally before sync.
- If offline, keep working; pending events sync when connection returns.
